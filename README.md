# pytorch-cifar10
Training model architectures like VGG16, GoogLeNet, DenseNet etc. on CIFAR-10 dataset
Any model listed in the code can be trained just by initiating the model function to the declared variable 'net'



# Model         Accuracy
LeNet         73.53

VGG16         91.47

GoogLeNet     92.93

DenseNet121   93.51
